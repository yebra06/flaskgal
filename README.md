# flaskgallery
Template rendering with flask.
## Instructions to get going
* Set up a good 'ol [virtualenv](https://virtualenv.pypa.io/en/stable/).
* Run the following on a unix-like command line.
* On terminal copy and paste the following:
```
$ git clone https://github.com/yebra06/flaskgallery.git
$ cd flaskgallery
$ pip install -r requirements.txt # assuming you have pip and virtalenv
$ python manage.py
```
